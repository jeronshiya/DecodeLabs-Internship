## Overview

This project focuses on creating a low-fidelity wireframe for an e-commerce landing page. The objective is to improve the user experience by designing a clean and logical page structure that helps users navigate easily and complete their shopping journey without confusion.

## Scenario

An e-commerce brand is losing customers during the checkout process due to a cluttered layout and poor user flow. The company requires a clear and organized landing page structure that guides users effectively.

## Objective

* Design the skeleton structure of the application
* Improve user flow and navigation
* Establish a clear visual hierarchy
* Create prominent Call-to-Action (CTA) buttons
* Reduce user confusion during the shopping process

## Wireframe Features

### Header

* Logo Placement
* Navigation Menu
* Search Bar
* Cart Option

### Hero Section

* Main Headline
* Supporting Text
* Primary CTA Button

### Product Categories

* Organized Category Layout
* Easy Navigation

### Featured Products

* Product Card Placeholders
* Structured Grid Layout

### Call-to-Action (CTA)

* Shop Now
* Explore Products
* View Details

### Footer

* Contact Information
* Support Links
* Quick Navigation

## User Flow

Homepage

↓

Browse Categories

↓

View Product

↓
Add to Cart
↓
Checkout
↓

Order Confirmation

## Design Principles

* Low-Fidelity Wireframing
* Black, White, and Grey Only
* Focus on Structure and Functionality
* Clear Navigation Hierarchy
* User-Centered Design Approach

## Tools Used

* Figma
* Canva

## Outcome

The wireframe provides a clear blueprint for the future user interface. It improves navigation, highlights important actions, and creates a structured shopping experience before moving to high-fidelity design.
