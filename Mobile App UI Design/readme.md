# Project 3 - Mobile App UI Design

## Overview
This project focuses on designing a modern and user-friendly mobile application interface. The design process included creating a visual style guide and key application screens to ensure a consistent and intuitive user experience.

## Objective
To create an attractive and functional mobile app interface that enhances usability and provides a seamless user experience.

## Tools Used
- Figma
- Canva

## Deliverables

### 1. Style Tile
A visual guide defining the application's color palette, typography, buttons, icons, and overall design language.

### 2. Splash Screen
The initial screen displayed when users launch the application.

### 3. Product List Screen
A screen that displays available products in an organized and user-friendly layout.

### 4. Product Details Screen
A detailed view of a selected product including image, description, price, and related information.

## Outcome
Successfully designed a complete mobile application user interface with a consistent visual identity and improved user experience.
